Conect EU

Conect EU is a platform that allows users to publish and discover ride listings across Europe.

Privacy Policy

Last updated: June 2026

Conect EU respects your privacy and is committed to protecting your personal information. This Privacy Policy explains what data we collect, how it is used, and your rights as a user.

Information We Collect

When using the Conect EU application we may collect:

  • Email address (used for authentication)
  • Display name
  • Listings created by the user
  • Application preferences (such as language or currency settings)
  • Technical information required for the application to function

We do not collect sensitive information such as banking data or precise location data.

How We Use the Information

The collected information is used only to:

  • Create and manage user accounts
  • Allow users to publish and view ride listings
  • Maintain and improve the application
  • Provide user support

Data Storage

User data may be stored securely using trusted cloud infrastructure providers. Reasonable technical measures are used to protect the data.

Data Sharing

Conect EU does not sell or rent personal data to third parties. Some data may be processed by infrastructure providers necessary for the application to operate.

User Rights

Users may request:

  • Deletion of their account
  • Removal of stored personal data
  • Information regarding stored data
