Privacy Policy for the Conect EU mobile application

Last updated: June 2026

Conect EU is a platform that allows users to publish and discover ride listings across Europe. We respect your privacy and are committed to protecting your personal information. This Privacy Policy explains what data we collect, how it is used, and what rights you have as a user.

Privacy Policy

1. Information We Collect

When using the Conect EU application, we may collect:

  • Email address (used for authentication)
  • Display name
  • Listings created by the user
  • Application preferences (such as language, theme, or currency settings)
  • Technical information required for the application to function properly

We do not collect sensitive information such as banking data or precise user location.

2. How We Use the Information

The information we collect is used only to:

  • Create and manage user accounts
  • Allow users to publish, manage, and view ride listings
  • Maintain and improve the application
  • Provide user support
  • Enable essential application functionality

3. Data Storage

User data may be stored securely using trusted cloud infrastructure providers. Reasonable technical and organizational measures are used to help protect the data.

4. Third-Party Services

The app may use trusted third-party services such as authentication, cloud storage, or messaging providers required for application functionality.

5. Data Sharing

Conect EU does not sell or rent personal data to third parties. Some data may be processed by technical service providers only where necessary for the application to operate properly.

6. User Rights

Users may request:

  • Deletion of their account
  • Removal of stored personal data
  • Information regarding stored data
